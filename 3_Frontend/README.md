
# 🚀 Resume Analyzer AI - Frontend

A modern AI-powered Resume Analyzer web application built with **React**. Users can upload their resumes, receive AI-generated job role predictions, and discover the latest job opportunities based on their skills and experience.

---

# 📌 Project Status

**Version:** V1.0

**Status:** 🟡 Under Development

---

# 🎯 Project Goal

The goal of this frontend is to provide a clean, modern, and interactive interface where users can:

- Upload Resume (PDF/DOCX)
- Analyze Resume
- View Top 5 Predicted Job Roles
- View Confidence Scores
- View Latest Job Opportunities
- Apply for Jobs
- Refresh Predictions

This version focuses only on the Resume Analyzer. Authentication, dashboard, and user profiles will be added in future versions.
